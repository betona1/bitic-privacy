LibroPrinter 플러그인 개인정보처리방침

주식회사 비트정보통신(BITIC Ltd., 이하 "회사")은 「LibroPrinter 플러그인」(이하 "본 앱")의 개인정보를 다음과 같이 처리합니다.

최종 개정일: 2026-04-29

1. 수집하는 개인정보 항목

본 앱은 어떠한 개인정보도 수집하지 않습니다. 회원가입, 로그인, 기기 식별자(광고 ID 등) 수집이 없습니다.

2. 인터넷 권한의 사용 목적

본 앱은 다음 목적으로만 INTERNET 권한을 사용합니다.

  • 같은 Wi-Fi 네트워크 내 LibroPrinter 키오스크 프린터 검색 (mDNS / Bonjour 프로토콜)
  • 사용자가 명시적으로 인쇄를 실행할 때, 인쇄할 PDF 데이터를 해당 키오스크로 전송 (IPP 프로토콜)

외부 서버(인터넷)와의 통신은 일절 발생하지 않으며, 모든 데이터 전송은 사용자의 로컬 네트워크 내부에서 이루어집니다.

3. 사용하지 않는 항목

항목사용 여부
광고 SDK사용 안 함
분석/통계 SDK (Firebase, Google Analytics 등)사용 안 함
크래시 리포팅 SDK사용 안 함
위치 정보 (GPS, 네트워크)사용 안 함
카메라 / 마이크 / 연락처 / 사진 / 캘린더사용 안 함
광고 ID (AAID)사용 안 함
전화번호 / 이메일 / 이름수집 안 함

4. 인쇄 데이터 처리

사용자가 인쇄를 실행하면, 안드로이드 시스템이 생성한 PDF가 사용자가 선택한 LibroPrinter 키오스크로 전송됩니다. 이 데이터는:

  • 본 앱의 메모리에 일시적으로만 존재하며, 인쇄 완료 시 즉시 폐기됩니다.
  • 본 앱이 별도로 저장하거나 외부로 전송하지 않습니다.
  • 키오스크가 인쇄 후 자체적으로 처리하며, 그 처리 방식은 키오스크 운영자의 정책을 따릅니다.

5. 만 14세 미만 아동의 개인정보

본 앱은 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.

6. 개인정보 처리방침의 변경

본 방침이 변경되는 경우, 변경 사항은 본 페이지에 게시되며 변경일을 명시합니다. 중대한 변경 시 앱 업데이트를 통해 사용자에게 고지합니다.
