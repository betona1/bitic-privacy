LibroPrinter Plugin — Privacy Policy

BITIC Ltd. (비트정보통신, "we", "us", or "the Company") operates the LibroPrinter Plugin Android application ("the App"). This Privacy Policy describes how the App handles user information.

Last updated: 2026-04-29

1. Personal Data Collected

The App does not collect any personal data. There is no account creation, no login, and no collection of device identifiers (advertising ID, IMEI, etc.).

2. Use of the INTERNET Permission

The App requests the INTERNET permission solely for the following purposes:

  • Discovering LibroPrinter kiosk printers on the same Wi-Fi network using mDNS (Bonjour) protocol.
  • When the user explicitly initiates a print job, transmitting the PDF document to the selected kiosk over the IPP protocol.

No communication with external servers occurs. All data transmission stays within the user's local network.

4. Print Data Handling

When the user prints, the Android system generates a PDF that the App transmits to the selected LibroPrinter kiosk. This data:

  • Exists only temporarily in memory and is discarded immediately after the print job completes.
  • Is not stored or transmitted externally by the App.
  • Is processed by the kiosk after receipt according to the kiosk operator's policy.

5. Children's Privacy

The App does not knowingly collect personal data from children under 14 years of age.

6. Changes to This Policy

If this policy changes, the updated policy will be posted on this page with a revised "Last updated" date. Material changes will be notified through an app update.
